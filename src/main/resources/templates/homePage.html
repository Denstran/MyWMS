Warehouse Management System WMS

Тестовый проект с использование Java Spring Framework и шаблонизатора Thymeleaf

Учебный проект по создание простой WMS

WMS позволит добовлять товары, оформлять их доставку, убирать их, а также просматривать товары
и сортировать их различными способами.
